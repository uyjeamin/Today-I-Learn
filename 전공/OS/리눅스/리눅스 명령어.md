```
{프로그램} > {파일} 
프로그램의 결과물을 파일에 저장(미리 저장되어 있는 것이 있으면 덮어 씌움)

ex:
cat file1 > file2 (file1 내용이 file2 로 감)\
echo "안녕" > file2 (안녕이 file2 에 저장됨.)

{프로그램} >> {파일}
프로그램의 결과물을 파일에 저장(뒤에 덧붙임)
ex:
echo "안녕" > file2 (안녕이 file2 에 저장됨.)
echo "하세요" >> file2 (안녕하세요가 file2 의 내용이 됨)

{프로그램} < {파일}
파일 내용을 프로그램에 입력
ex:
```
