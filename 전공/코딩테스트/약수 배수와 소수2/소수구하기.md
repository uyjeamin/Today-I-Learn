

## 부르투포스 방식
>[!note] 파라미터로 값 한개씩 받아서 그것이 소수인지 판단하기 좋음

```java
	
	public static long prime(long x){
	    int su = (int)Math.sqrt(x);
	    
	    if(x==2 || x==3){
	        return x;
	    }else if(x==1){
	        return -1;
	    }
	    
	    for(int i=2 ; i<=su ; i++){
	        if(x%i==0){
	            return -1;
	        }
	    }
	    return x;
	}
```
